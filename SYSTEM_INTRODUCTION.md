# e-Regi 시스템 소개서

## 시스템 개요

**e-Regi**는 학술 단체 및 컨퍼런스를 위한 종합 이벤트 관리 플랫폼입니다. 등록, 결제, 출석 관리, 배지 발급, 알림톡 발송 등 이벤트 운영에 필요한 모든 기능을 하나의 시스템에서 통합 관리할 수 있습니다.

### 🎯 핵심 가치

- **통합 관리**: 회원 관리, 행사 등록, 결제, 출결, 배지 발급까지 원스톱 처리
- **다중 테넌트**: 여러 학술 단체(Society)와 컨퍼런스를 단일 플랫폼에서 운영
- **모바일 최적화**: QR코드 기반 출결 시스템으로 빠르고 정확한 참가자 관리
- **실시간 모니터링**: 라이브 대시보드로 실시간 참가 현황 파악

---

## 💡 왜 e-Regi인가?

### 🚀 기존 시스템의 문제점
대부분의 학술 단체와 컨퍼런스 운영진들은 다음과 같은 문제를 겪고 있습니다:

- **여러 시스템 병행**: 등록은 홈페이지, 결제는 PG사, 출결은 엑셀, 배지는 별도 프로그램
- **수작업 반복**: 참가자 정보를 여러 시스템에 수동으로 입력하는 비효율
- **현장 혼란**: 종이 명부로 출결 체크하여 기다림과 오류 발생
- **실시간 파악 불가**: 행사 현장의 참가 현황을 즉시 확인하기 어려움
- **데이터 분산**: 결제, 등록, 출결 데이터가 각기 다른 곳에 저장되어 통합 분석 불가

### ✨ e-Regi만의 차별점

#### 1. **완전한 통합 플랫폼**
```
등록 → 결제 → 출결 → 배지 → 통계
모든 과정이 하나의 시스템에서 연동됩니다.
```
- **시스템 간 데이터 이동 없이** 모든 프로세스가 자동으로 연결
- 참가자 등록 정보가 그대로 배지로, 출결 데이터로 자동 변환
- 수작업으로 인한 인건비와 오류를 획기적으로 감소

#### 2. **QR코드 기반 스마트 출결**
```
기존: 종이 명부 서명 → 1인당 30초 이상 소요
e-Regi: QR코드 스캔 → 1인당 3초 이내 완료
```
- **10배 빠른 입장 처리**: 대기 시간 최소화로 참가자 만족도 향상
- **실시간 출결 현황**: 누가 왔는지, 몇 명이 참여했는지 즉시 파악
- **중복 체크 방지**: 같은 사람이 여러 번 입장하는 것을 자동으로 방지
- **정확한 데이터**: 인식 오류로 인한 출결 오류 제로화

#### 3. **학술 단체 특화 기능**
```
학회 회원 검증 → 회원 할인 자동 적용 → 회원증 발급
```
- **회원 등급별 등록비 자동 계산**: 정회원, 준회원, 학생회원 등급별로 다른 요금 자동 적용
- **회원증번호 검증**: 회원 데이터베이스와 연동하여 회원 여부 즉시 확인
- **다중 학회 지원**: 하나의 플랫폼에서 여러 학회 행사를 각각 운영 가능

#### 4. **실시간 모니터링 대시보드**
```
행사 당일:
- 현재 참가자 수: 234명 / 300명 (78%)
- 체크인 완료: 198명
- 미도착: 36명 (알림톡 자동 발송 가능)
```
- **현장 상황 한눈에 파악**: 별도의 리포팅 없이 실시간 데이터 확인
- **데이터 기반 의사결정**: 참가율이 낮은 세션에 즉각적인 대응 가능
- **스태프 배치 최적화**: 붐비는 시간대와 게이트를 미리 파악하여 인력 배치

#### 5. **자동화된 커뮤니케이션**
```
등록 → 알림톡 자동 발송
결제 → 영수증 이메일 자동 발송
배지 준비 완료 → 알림톡 자동 발송
```
- **운영진의 업무 시간 절감**: 수동으로 메일/메시지 보내는 시간을 90% 이상 단축
- **참가자 경험 향상**: 즉각적인 확인으로 안심하고 행사 참여
- **누락 방지**: 사람이 하는 일이 아니므로 실수로 알림을 못 보내는 일이 없음

#### 6. **확장 가능한 모듈 구조**
```
기본: 등록 + 결제 + 출결
옵션: 배지 시스템, 후원사 관리, 프로그램 관리, 알림톡
```
- **필요한 기능만 선택**: 조직의 규모와 예산에 맞춰 기능 선택 가능
- **단계적 도입 가능**: 작은 행사부터 시작하여 점차 기능 확장
- **맞춤화 유연**: 각 학회/컨퍼런스의 특성에 맞춰 설정 가능

#### 7. **학회(Society) 전용 포털**
```
기존: 각 행사마다 별도 웹사이트 제작
e-Regi: 학회별 자동 포털 + 모든 행사 통합 관리
```
- **학회별 독립 웹사이트**: `kadd.eregi.co.kr` 형태의 전용 도메인 자동 생성
- **일관된 브랜딩**: 학회 로고, 색상, 콘텐츠로 학회 정체성 유지
- **회원 데이터베이스**: 학회 회원 정보를 모든 행사에서 재사용
- **마이페이지 제공**: 회원이 자신의 정보, 등록 현황, 참가 이력을 한 곳에서 관리
- **다중 학회 지원**: 하나의 플랫폼에서 여러 학회를 각각 운영 가능

#### 8. **파트너(밴더) 지원 시스템**
```
기존: 종이 명부에 사인 → 나중에 데이터 입력 → 엑셀로 공유
e-Regi: QR코드 스캔 → 자동 리드 수집 → 파트너 포털에서 즉시 확인
```
- **파트너 만족도 향상**: 후원사가 자체 포털에서 실시간 방문자 데이터 확인
- **리드 수집 자동화**: 종이 명부 없이 QR코드 스캔만으로 리드 자동 수집
- **데이터 소유권 명확**: 파트너가 자신의 데이터를 직접 관리하고 내보내기
- **스태프 협업 지원**: 여러 스태프가 동시에 스캔하며 실시간 데이터 공유
- **스탬프 투어 연동**: 참가자 참여 유도와 파트너 방문 활성화

### 📊 도입 효과

#### 운영 효율성 측면
- **인건비 절감**: 연간 4회 행사 운영 시 약 200시간 이상 절감
- **오류 감소**: 수작업으로 인한 데이터 오류 99% 이상 감소
- **업무 속도**: 참가자 등록 확인 시간 10분에서 10초로 단축

#### 참가자 만족도 측면
- **입장 대기 시간**: 평균 5분에서 30초 이내로 단축
- **정보 접근성**: 행사 정보, 내 배지, 내 출결 현황을 언제든 확인
- **실시간 소통**: 행사 중 알림톡으로 중요 공지 즉시 전달

#### 데이터 활용 측면
- **통합 분석**: 등록, 결제, 출결 데이터를 연결하여 참가자 행동 패턴 분석
- **다음 행사 기획**: 어느 세션이 인기 있는지, 어떤 시간대가 붐비는지 데이터로 증명
- **회원 관리**: 누가 우리 학회 행사에 자주 참여하는지 파악하여 회원 관리에 활용

## 주요 기능

### 1. 사용자 관리

#### 🔐 통합 인증 시스템
- **Firebase Authentication** 기반 보안 인증
- 이메일/소셜 로그인 지원
- 역할별 접근 제어 (Super Admin, Society Admin, Staff, User)
- 외부 참가자를 위한 임시 인증 토큰 발급

#### 👥 회원 등급 관리
- 학회 회원 등급별 등록비 설정 (정회원, 준회원, 학생회원 등)
- 회원증번호로 회원 여부 자동 검증
- 회원 할인 적용 및 비회원 요금 차등 부과

### 2. 행사 등록 시스템

#### 📝 유연한 등록 폼
- **기본 정보 수집**: 성명, 소속, 이메일, 연락처
- **옵션 선택**: 점심식사, 만찬, 세부 프로그램 참가 신청
- **부가 옵션**: 추가 상품/서비스 선택 및 결제
- 다국어 지원 (한글/영문)

#### 💳 통합 결제 시스템
- **NICEPAY** 결제 연동
- **토스페이먼츠** 결제 연동
- 실시간 결제 승인 및 금액 검증
- 결제 실패/취소 처리 및 환불 기능
- 영수증 자동 발급

### 3. 출결 관리 시스템

#### 📱 QR코드 기반 출결
- **참가자**: 개별 QR코드 배지로 스캔하여 입장
- **스태프**: 모바일 스캐너로 빠른 QR코드 스캔
- **게이트**: 입구별 QR체크 시스템
- **정보석**: 참가자 조회 및 수동 체크인/아웃

#### ⚡ 실시간 출결 현황
- **라이브 대시보드**: 실시간 참가자 현황 모니터링
- **일괄 퇴장**: 행사 종료 시 전원 일괄 체크아웃
- **중복 스캔 방지**: 10초 내 중복 스캔 방지 로직
- **통계**: 세션별/전체 참가율 실시간 집계

### 4. 배지 시스템

#### 🎨 커스텀 배지 디자인
- 드래그앤드롭 배지 에디터
- 이름, 소속, QR코드, 사진 등 요소 배치
- 다양한 템플릿 제공
- 실시간 미리보기

#### 🖨️ 다양한 출력 옵션
- **프린터 출력**: Zebra/Sato 휴대용 프린터 지원
- **PDF 다운로드**: 개별/일괄 PDF 생성
- **디지털 배지**: 모바일용 디지털 배지 발급
- **알림톡 발송**: 배지 준비 완료 알림 자동 발송

### 5. 알림 및 커뮤니케이션

#### 📨 알림톡 서비스
- **NHN Cloud 알림톡** 연동
- 등록 완료, 배지 발송, 행사 알림 자동 발송
- 대량 발송 기능 지원
- 발송 결과 모니터링

#### 📧 이메일 시스템
- 등록 확인 이메일 자동 발송
- 결제 영수증 이메일 전송
- 맞춤형 이메일 템플릿 지원

### 6. 행사 콘텐츠 관리

#### 📋 프로그램 관리
- 행사 일정(Agenda) 관리
- 연사자 정보 관리
- 프로그램 세션별 구성
- 다국어 프로그램 지원

#### 🏢 후원사 관리
- 후원사 등록 및 관리
- 후원사 로고 및 소개 표시
- 후원사별 부스 관리
- 협찬 내역 관리

### 7. 학회(Society) 전용 포털

#### 🏛️ 학회별 독립 웹사이트
- **자동 생성되는 학회 페이지**: 각 학회별로 독립적인 랜딩 페이지 제공
- **서브도메인 지원**: `kadd.eregi.co.kr` 형태의 학회 전용 도메인
- **다국어 지원**: 한글/영문 바이링귀얼 콘텐츠 관리
- **학회 브랜딩**: 로고, 컬러, 소개 등 학회 고유의 아이덴티티 표현

#### 📋 학회 콘텐츠 관리
- **회장 인사말**: 학회장 소개와 비전 제시
- **학회 소개**: 역사, 비전, 조직 구조 등 상세 정보
- **공지사항 관리**: 학회 주요 소식 및 알림 발행
- **행사 목록**: 학회 주최 모든 컨퍼런스 일괄 관리
- **마이페이지**: 회원 정보, 등록 현황, 참가 이력 확인

#### 🎯 학회 회원 관리

### Frontend
- **React 19.2** + **TypeScript 5.9**
- **Vite** (Rolldown 기반 빌드)

### Backend
- **Firebase** (백엔드 서비스)
  - **Firestore**: 문서 지향 데이터베이스
  - **Firebase Authentication**: 사용자 인증
  - **Cloud Functions**: 서버리스 API
  - **Firebase Hosting**: 정적 웹 호스팅
- **Node.js 20** (Cloud Functions 런타임)

### 결제 시스템
- **NICEPAY**: PG사 연동
- **토스페이먼츠**: 결제 위젯 연동

- **NHN Cloud 알림톡**: 카카오톡 알림톡 발송

### QR코드 & 배지
- **html5-qrcode**: QR코드 스캐닝
- **qrcode.react**: QR코드 생성
- **html2canvas**: 배지 이미지 변환
- **jsPDF**: PDF 생성

### 테스트 & 품질
- **Jest**: 단위 테스트
- **Playwright**: E2E 테스트
- **ESLint**: 코드 린팅
- **Husky**: Git hooks 자동화
