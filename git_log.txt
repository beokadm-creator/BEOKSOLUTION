commit d554ce4adcde4866fa22fccc2351f5d9c1e631f3
Author: Aaron <beokadm@gmail.com>
Date:   Mon Feb 9 13:16:41 2026 +0900

    fix: prevent TypeError when g.name is undefined in RegistrationPage
    
    Fix the 'e.name.toLowerCase is not a function' error that occurred 9 times
    at https://kadd.eregi.co.kr/2026spring/register.
    
    Root cause:
    - grades array elements may have undefined 'name' property
    - Calling .toLowerCase() on undefined throws TypeError
    
    Solution:
    - Changed: const n = g.name.toLowerCase();
    - To: const n = String(g.name || '').toLowerCase();
    - This safely handles undefined/null values by converting to empty string
    
    This is a defensive programming fix that prevents runtime errors
    when grade data structure is incomplete or malformed.
