## 📅 T-7 (1주 전)

### 전체 함수 배포 및 스모크 테스트

- [ ] **함수 컴파일**: `cd functions && npm run build` 실행 (컴파일 에러 없어야 함)
- [ ] **전체 함수 배포**: `firebase deploy --only functions --project eregi-8fc1e`
- [ ] **함수 생존 확인**: `firebase functions:list --project eregi-8fc1e`로 45개 함수 모두 배포 확인
  - [ ] `docs/functions_snapshot_post_cleanup_final.txt`와 비교하여 누락 함수 없는지 확인
  - [ ] 주석 처리된 export 문이 없는지 확인 (`functions/src/index.ts` 확인)

### 핵심 경로 스모크 테스트 (E2E)

- [ ] **등록 플로우**: 테스트 등록 생성 → Firestore 데이터 확인 → 결제 확인
- [ ] **QR 스캔**: 테스트 QR 코드 생성 → 스캐너 앱으로 스캔 → 입장 로그 확인
- [ ] **배지 발급**: 인포데스크에서 테스트 배지 프린트 (프린터 연결 확인)
- [ ] **알림톡 발송**: 테스트 알림톡 발송 → 수신 여부 확인
- [ ] **번역 시스템**: 테스트 세션에서 번역 패널 렌더링 확인

### 인프라 동기화

- [ ] **인덱스 동기화**: `firebase firestore:indexes --project eregi-8fc1e > firestore.indexes.json` 실행 후 비교
- [ ] **보안 룰 배포**: `firebase deploy --only firestore:rules --project eregi-8fc1e`
- [ ] **번역 시스템 확인**: DeepL API 키 및 세션 필터 로직 확인
