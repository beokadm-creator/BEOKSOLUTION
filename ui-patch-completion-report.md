## 🚀 사용 가이드

### 관리자 페이지 접속
1. `/admin/conference/registrations` 접속
2. 페이지 우측 상단에 새 버튼 2개 확인

### 무결성 검사 사용
1. **[📊 무결성 검사]** 버튼 클릭
2. 자동으로 데이터 정합성 검사 실행
3. 완료 후 결과 알림창 표시

### 고아 데이터 정리 사용
1. **[🧹 고아 데이터 정리]** 버튼 클릭
2. 확인창에서 "확인" 클릭
3. 삭제된 회원의 잔여 로그 정리
4. 완료 후 삭제 개수 알림창 표시
